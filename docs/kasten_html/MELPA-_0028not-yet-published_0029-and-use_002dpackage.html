5.1 MELPA (not yet published) and use-package ¶

(unpublished) If MELPA is in the package archives, simply:

M-x package-install RET kasten RET

If using ‘use-package’, add the following to the Emacs configuration file (such as ‘init.el’): 

(use-package kasten
  :ensure t
  :defer t
  :commands (kasten))
