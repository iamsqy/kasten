9.2 Reporting Security Concerns ¶

If you found any vulnerabilities or would like to report any security concerns, please do not use any public channels. Please contact the package author via email. Please sign and encrypt your email with PGP1. Encrypted plain text email MIMEs are preferred (do not use multipart/alternative to include HTML copies of the email).
