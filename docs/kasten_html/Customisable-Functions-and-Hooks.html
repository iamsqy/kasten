7.2 Customisable Functions and Hooks ¶

If the ID contains title (for example, using ‘§250229-23 332. 2. -foo’ as ID of the note created on 2025-02-29 at 23:33 titled ‘foo’), one can customise how to convert the note title to a safer string which can be used as part of the filename.

Variable: kasten-safetitle-function ¶

Function to convert a title to a safe short string.

Default value: ‘kasten--default-safetitle’.

Function: kasten--default-safetitle ¶

Default safetitle function to convert TITLE to max 8 char string.
